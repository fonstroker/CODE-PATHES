Базовые типы данных:
1) int - целочисленный тип (ToInt32)
2) double - вещественный тип (ToDouble)
3) bool - логический тип (ToBoolean)
4) char - символьный тип (ToChar)
5) string - строковый тип (ToString) (Length - количество символов в строке)
6) object - универсальный тип (Object)Для работы с переменными их необходимо объявить:
<тип данных> <имя переменной>;
Присвоение значения переменной:
<имя переменной> = <значение>;
У различных типов данных есть разные правила присвоения:
Для double (вещественных типов) через точку, для string в двойных ковычках, для char в одинарных ковычках.Преобразование типов данных
Иногда, для работы с переменными, их необходимо преобразовать:
Convert.To<Тип преобразования> (Он указан в скобках возле типов данных)

List - это универсальная коллекция, которая представляет собой динамический массив. List позволяет хранить набор элементов одного типа в порядке их добавления. Тип bool не может использоваться.
List используется для:
1. Хранения и управления набором объектов одного типа.
2. Динамического добавления и удаления элементов.
3. Индексированния доступа к элементам.Конструкция List:
List<T>, где T - тип элементов, которые вы хотите хранить.Функции List:
1. Add(T item) - добавляет элемент в конец списка.
2. Remove(T item) - удаляет первое вхождение указанного элемента.
3. RemoveAt(int index) - удаляет элемент по индексу.4. Count - возвращает количество элементов в списке.
5. Contains(T item) - проверяет наличие элемента в списке.
6. Clear() - удаляет все элементы из списка.

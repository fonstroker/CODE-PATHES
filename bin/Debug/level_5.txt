Функция - это подпрограмма, куда можно вынести повторяющийся код и обращаться к нему, когда это будет нужно. Для того, чтобы вернуть из функции значение, необходимо использовать служебное слово return. Возращаемое значение должно соответствовать типу возращаемого значения.Конструкция:
public static <тип возращаемого значения> <имя функции> (<параметры>) 
{
<тело функции>
}Тип возращаемого значения - это то, что будет возвращать функция после ее вызова.
Типы возвращаемого значения:
string - возращает строку;
int - возвращает целое число;
char[] - символьный массив;
...
void - ничего не возвращает.Параметры - это значения, которые принимает функция. Можно оставить скобки пустыми, тогда функция не будет ничего принимать.

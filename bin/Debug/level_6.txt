Запись в текстовый файл осуществляется с помощью класса StreamWriter и свойством AppendText.
Общая конструкция записи текста в конец файла:
using (StreamWriter <имя переменной> = File.AppendText(<путь к файлу>))
{
<имя переменной>.WriteLine(<Содержимое записи>);
}Чтение информации из файла может осуществлятся несколькими способами:
1) StreamReader;
2) File.ReadAllLines;
3) File.ReadAllText.StreamReader - это класс, который позволяет читать текст из файла.
Например:File.ReadAllLines - это метод используется для чтения всех строк из файла в массив строк.
Например:File.ReadAllText - это метод позволяет прочитать содержимое файла и вернуть его в виде строки.

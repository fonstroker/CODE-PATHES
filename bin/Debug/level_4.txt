Массив - это последовательность упорядоченных типизированных элементов одного типа данных, которые хранятся под одним именем и обращаются по индексу, который начинается с 0.
Конструкция:
<тип> [] <имя> = new <тип> [<длина>];Можно объявить массив другим способом - не задавая его размерность, а указывая элементы массива сразу при его объявлении в фигурных скобках:
<тип> [] <имя> = {елемент 1, ...};
Функция Length - возвращает длину массива.Двумерный массив - массив элементов расположенных в виде таблицы с двумя измерениями - строками и столбцами.
Конструкция:
<тип> [,] <имя>= new <тип> [<количество строк>, <количество столбцов>];Пример заполнения одномерного массива с помощью цикла for:Пример заполнения двумерного массива с помощью двойного цикла for:Простая сортировка одномерного массива от наименьшего к большему:

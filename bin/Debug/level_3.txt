Операторы ветвления делятся в основном на if и switch.
Конструктрукция оператора ветвления if:
if (<условие>)
{
<тело 1>
}
else
{
<тело 2>
}Конструкция оператора ветвления switch:
switch (<выражение>)
{
case <значение 1>: <тело 1>; break;
case <значение 2>: <тело 2>; break;
default: <тело 3>; break;
}Основные циклы: while, for и foreach.
Цикл while делится на два вида - с постусловием и предусловием. Различия между этими циклами в том, что цикл с постусловием выполнится хотя бы один раз точно, а с предусловием нет.
Конструкция цикла while с предусловием:
while (<условие>)
{
<тело>
}Конструкция цикла while с постусловием:
do
{
<тело>
} while (<условие>);Конструкция цикла с параметром for:
for (<начальное значение>; <условие>; <шаг>)
{
<тело>
}Конструкция цикла перебора коллекций foreach:
foreach (<тип> <переменная> in <коллекция>)
{
<тело>
}

CODE PATHES - это Desktop-приложение на C# WPF, которое превращает изучение программирования в увлекательное игровое путешествие по различным "тропам" знаний с интерактивными заданиями и визуальным представлением прогресса. CODE PATHES создает персонализированный игровой опыт обучения, который позволяет:
Изучать C# через визуальные интерактивные задания

Отслеживать прогресс в виде карты путешествия

Получать мгновенную обратную связь при выполнении упражнений

Осваивать программирование в своем темпе
